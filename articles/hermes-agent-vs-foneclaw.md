## What Is Hermes Agent?

Hermes Agent is an open-source AI agent framework that supports multiple AI models. It's designed for developers and enterprises who want to build custom AI agents.

**Key features:**
- Multi-model support (Claude, GPT, Gemini, local models)
- Skill-based architecture for customization
- Gateway system for multi-platform deployment
- Active community and documentation

**Best for:**
- Developers building custom AI agents
- Enterprises needing flexible automation
- Research and experimentation
- Multi-model orchestration

## What Is FoneClaw?

FoneClaw is a specialized AI agent for hands-free Android phone control. It's designed for users who want to control their phone entirely by voice.

**Key features:**
- 50+ voice operations
- Multi-step automation
- Works with any Android app
- Local processing for privacy

**Best for:**
- Hands-free phone control while driving
- Elderly users who struggle with smartphones
- Accessibility for visually impaired users
- Cooking, exercising, or any hands-busy situation

## Frequently Asked Questions

**Q: Can Hermes Agent control my Android phone?**
A: Hermes Agent is a general AI platform, not specifically designed for phone control. For hands-free Android control, FoneClaw is the better choice.

**Q: Is FoneClaw open-source like Hermes Agent?**
A: No, FoneClaw is a commercial product with a free tier. Hermes Agent is fully open-source.

**Q: Which has better privacy?**
A: Both offer strong privacy. Hermes Agent can be self-hosted, while FoneClaw processes everything locally on your device.

**Q: Can I use Hermes Agent on my phone?**
A: Hermes Agent is designed for server deployment, not direct phone use. FoneClaw is built specifically for Android phones.

**Q: Which is easier to set up?**
A: FoneClaw is much easier—just install the app. Hermes Agent requires technical knowledge for server setup and configuration.
