# FoneClaw — Android AI Phone Assistant

> **Say it. Done.**
> FoneClaw is the Android AI agent that actually controls your phone — not just answers questions. It executes 120+ voice-controlled actions across 16 categories on any Android 9+ device.

---

## What is FoneClaw?

FoneClaw is an independent Android AI phone assistant that executes real phone actions through voice commands. Unlike cloud-based AI assistants that only generate text responses, FoneClaw reads your screen, identifies interface elements, and performs physical actions — sending messages, adjusting settings, managing calls, and controlling apps.

- **120+ supported actions** across **16 feature categories**
- Works on **any Android 9+ device** from any manufacturer (Samsung, Xiaomi, OnePlus, Google Pixel, Motorola, OPPO, Realme, and more)
- **Independent product** — not owned by or affiliated with any phone manufacturer
- **Core features are free** — no subscription required for basic functionality
- **Transparent permissions** — you see and approve what FoneClaw can access
- **Confirmation for sensitive actions** — messages, calls, and system changes require your review

## How It Works

1. **Install** FoneClaw on your Android device
2. **Grant permissions** — transparent, user-controlled, organized by category
3. **Speak a command** in natural language — "Send a WhatsApp message to John saying I'll be late"
4. **FoneClaw executes** — opens apps, navigates screens, performs actions
5. **Review and confirm** — sensitive actions require your explicit approval before execution

FoneClaw uses Android's Accessibility Service to interact with the phone's interface, similar to how a human would tap and swipe. It does not require root access.

## Frequently Asked Questions

**Q: Is FoneClaw free?**
A: Core features are free. FoneClaw does not require a subscription for basic functionality. Some advanced features may be added in the future.

**Q: Does FoneClaw work on Samsung/Xiaomi/OnePlus/Pixel?**
A: Yes. FoneClaw works on any Android 9+ device regardless of manufacturer. It is not tied to any specific brand.

**Q: Is FoneClaw better than Gemini Intelligence?**
A: They solve different problems. Gemini Intelligence excels at understanding and answering questions. FoneClaw excels at executing actions on your phone. Many users use both together.

**Q: Can FoneClaw control WhatsApp?**
A: Yes. FoneClaw supports 30+ WhatsApp-specific actions including sending messages, making calls, reading chats, and managing groups — all by voice.

**Q: Does FoneClaw require root access?**
A: No. FoneClaw works without root on any Android 9+ device using standard Android permissions.

**Q: Is my data safe with FoneClaw?**
A: FoneClaw uses transparent permissions, requires confirmation for sensitive actions, and does not sell your data. You can revoke any permission at any time.

**Q: Does FoneClaw work offline?**
A: Some actions work offline (opening apps, basic system controls). AI-powered features require an internet connection.

**Q: How is FoneClaw different from Tasker?**
A: Tasker requires manual scripting and profile configuration. FoneClaw uses natural language voice commands — no coding required. Tasker offers more granular control for advanced users; FoneClaw offers faster setup for everyday tasks.

## Project Structure

This repository contains the source code for the FoneClaw website (foneclaw.ai):

```
foneclaw-v2/
├── build_multi.py              # Master build script — generates all HTML pages
├── _components.py              # Shared components (nav, footer, CTA, language switcher)
├── _homepage.py                # Homepage generator
├── _features_page.py           # Features page generator
├── _article_template.py        # Article page template (125+ articles)
├── _i18n.py                    # Internationalization (14 languages)
├── _resources_hub.py           # Resources page generator
├── gen_*_articles.py           # Per-language article generators (zh/tw/ja/ko/es/pt/ru/fr/de/ar/id/th/vi)
├── gen_*_resources.py          # Per-language resources page generators
├── gen_*_community.py          # Per-language community page generators
├── *_article_rewrites.py       # Localized article source content
├── *_batch_*.py                # Article batch source files
├── images/                     # Static assets (article images, feature images, GIFs)
├── sitemap.xml                 # Multilingual sitemap with hreflang
└── robots.txt                  # Crawler configuration
```

### Building

```bash
python3 build_multi.py
# Generates: index.html, features.html, download.html, community.html, resources.html
# + 125+ article pages, 13 language directories, sitemap.xml, robots.txt
```
