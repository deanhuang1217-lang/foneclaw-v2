## Technical Implementation

- **URL structure**: Subdirectory-based (`/zh/`, `/ja/`, `/es/`, etc.)
- **Hreflang**: Full cross-referencing between all 14 languages + x-default
- **Language switcher**: Absolute URLs, 14 options, data-lang attribute for bfcache handling
- **Sitemap**: 354 URLs with hreflang cross-references
- **RTL support**: Arabic (`/ar/`) uses `dir="rtl"`
- **CTA localization**: APK download labels in all 14 languages
- **Footer**: Localized per language, no hub page links in non-English versions

## Content Quality

- **Not machine-translated**: Each language version is written by native speakers or reviewed for natural expression
- **well-structured**: Each article follows content best practices (H2 structure, FAQ, internal links, E-E-A-T signals)
- **Conservative product claims**: No "free forever", no "works with every app", no unverified superiority claims
- **Localized E-E-A-T**: Each language uses natural first-person experience signals in the local language
