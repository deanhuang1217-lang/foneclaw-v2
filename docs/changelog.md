# FoneClaw Changelog

## v1.0.0 — Pre-release (2026)

### Features
- 120+ voice-controlled phone actions across 16 categories
- Natural language voice command processing
- Cross-app automation (WhatsApp, Telegram, SMS, email, calendar, etc.)
- Phone status reporting (battery, storage, network, permissions)
- Daily briefing (SMS, notifications, system info summary)
- Passive triggers (automatic screenshot/photo detection)
- Transparent permission model with per-category control
- Confirmation for sensitive actions
- No root required

### Multilingual Support
- 14 languages launched: English, Simplified Chinese, Traditional Chinese, Japanese, Korean, Spanish, Portuguese, Russian, French, German, Arabic, Indonesian, Thai, Vietnamese
- Each language includes 5 core pages + 12 localized articles
- Localized content (not machine-translated)

### Platform
- Android 9.0+ support
- Works on all Android manufacturers (Samsung, Xiaomi, OnePlus, Google Pixel, Motorola, OPPO, Realme, etc.)
- 3 GB RAM minimum, 100 MB storage

### Website
- Full multilingual website at foneclaw.ai
- 14 language versions with hreflang tags
- 125+ English articles, 12 featured articles per language
- Responsive design, dark theme
- APK download with version checking
