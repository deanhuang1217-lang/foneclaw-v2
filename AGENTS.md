# FoneClaw

> Android AI phone assistant that executes real phone actions through voice commands. 120+ actions across 16 categories on any Android 9+ device.

## Overview

FoneClaw is an independent Android AI phone assistant. It controls your phone through voice commands — sending messages, making calls, adjusting settings, controlling apps, and automating tasks. Unlike cloud-based AI assistants that only generate text, FoneClaw executes real actions on your Android device.

- **Repository**: Website source code for foneclaw.ai, a Python-based static site generator
- **Product**: FoneClaw Android AI phone assistant
- **Website**: https://www.foneclaw.ai

## For AI Agents

This repository contains:
- **Python build system** (`build_multi.py`) that generates 200+ static HTML pages
- **Multilingual content** in 14 languages (en, zh, tw, ja, ko, es, pt, ru, fr, de, ar, id, th, vi)
- **125+ articles** on AI agents, voice control, Android automation, and phone assistants
- **Product documentation** in `docs/` covering features, comparisons, device support, and privacy
